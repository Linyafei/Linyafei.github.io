Linux 常用命令
2020-06-18

你可以使用 man [命令] 来查看各个命令的使用文档，如 ：man cp。

文件管理
cat 由第一行开始显示文件内容
chmod 666 a.txt 修改文件权限
cmp 命令用于比较两个文件是否有差异。
diff 以逐行的方式，比较文本文件的异同处。
find 命令用来在指定目录下查找文件
file 命令用于辨识文件类型。
mv ./a.txt ../ 命令用来为文件或目录改名、或将文件或目录移入其它位置。
rm 命令用于删除一个文件或者目录。
touch 命令用于修改文件或者目录的时间属性，包括存取时间和更改时间。若文件不存在，系统会建立一个新的文件。
cp ./a.txt ../ 命令主要用于复制文件或目录。
read 命令用于从标准输入读取数值。

文档编辑
look A a.txt 查找A开头的单词。 命令用于查询单词。
expr 10 + 10 命令是一个手工命令行计数器，求表达式变量的值，一般用于整数值，也可用于字符串。
wc 命令用于计算字数。

磁盘管理
cd： 切换目录
ls: 列出目录及文件名cp: 复制文件或目录
pwd： 显示目前的目录
mkdir： 创建一个新的目录
rmdir： 删除一个空的目录

网络通讯
telnet 命令用于远端登入。
httpd 命令是Apache HTTP服务器程序。
ifconfig命令用于显示或设置网络设备。
netstat 命令用于显示网络状态。
netstat 命令用于显示网络状态。
ping命令用于检测主机。

系统管理
date 命令可以用来显示或设定系统的日期与时间
exit 命令用于退出目前的shell。
sleep 命令可以用来将目前动作延迟一段时间。
last 命令用于显示用户最近登录信息。
who 命令用于显示系统中有哪些使用者正在上面
sudo 命令以系统管理者的身份执行指令，也就是说，经由 sudo 所执行的指令就好像是 root 亲自执行。
su 命令用于变更为其他使用者的身份，除 root 外，需要键入该使用者的密码。
w 命令用于显示目前登入系统的用户信息。

系统设置
clear 命令用于清除屏幕。
enable 命令用于启动或关闭 shell 内建指令。
