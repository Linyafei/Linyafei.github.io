Flutter 混入 iOS
2020-04-12

首先在自己电脑上已经安装过Flutter，一般安装 Andriod Studio 在插件中搜索Flutter即可安装

如何把Flutter混入iOS项目中，实现Flutter和iOS混合编程

1 用Xcode创建一个iOS项目，如MyIOS
2 pod init 创建Podfile文件
3 flutter create -t module my_flutter (my_flutter为自己定义的Flutter名称，可以自己起名)
4 在Podfile文件中添加如下代码， 然后 pod install， 即可把 flutter集成到iOS项目中
flutter_application_path = ‘./my_flutter/‘
load File.join(flutter_application_path, ‘.ios’, ‘Flutter’, ‘podhelper.rb’)
target ‘MyIOS’ do
use_frameworks!
install_all_flutter_pods(flutter_application_path)
end
备注：
此处路径 ‘./my_flutter/‘ 为第3中创建my_flutter的路径，在MyIOS在文件夹下和Podfile在同一级文件中，也可自己修改文件路径，但要保证引用的是创建my_flutter时的路径

5 用Xcode打开项目引入 import Flutter 添加如下代码，即可调用Flutter
let flutterVC = FlutterViewController.init()
flutterVC.setInitialRoute(“routel”)
self.present(flutterVC, animated: true, completion: nil)

6 用Andriod Studio 打开my_flutter即可在编写Flutter，然后回到Xcode查看Flutter是否有所改变

值得一提的是 以前引入用的是
eval(File.read(File.join(flutter_application_path, ‘.ios’, ‘Flutter’, ‘podhelper.rb’)), binding)
现在是
load File.join(flutter_application_path, ‘.ios’, ‘Flutter’, ‘podhelper.rb’)
